StringBuffer class is used to create a mutable string.

Methods of StringBuffer class:

•	length(): The length() method returns the length of the StringBuffer.
•	capacity(): The capacity() method returns the current capacity of the StringBuffer. The default capacity of the buffer is 16.
•	append():This method is used to concatenate two strings.
•	insert(): The insert() method inserts the given string to the StringBuffer at the specified position.
•	reverse(): The reverse() method reverses the characters of the StringBuffer and returns it.
•	replace(): The replace() method replaces the StringBuffer object with the specified string. It accepts three parameters startIndex, endIndex and the string. The substring that is to be replaced is specified by the startIndex and endIndex.
•	delete(): The delete() method deletes the character sequence in the StringBuffer. It accepts two parameters startIndex and endIndex. The characters are deleted from the startIndex until endIndex-1.
•	deleteCharAt(): The deleteCharAt() method deletes the character in the StringBuffer at the specified index.
