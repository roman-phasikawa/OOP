Interface is similar to an abstract class.It's purpose to use multiple inheritance.

Notes:

1.Interface methods do not have a body the "implement class provides the body.

2.On implementation of an interface, you must override all of its methods.

3.Interface methods are by default abstract and public.

4.Interface attribute are by default public, static and final.

5.An interface cannot contain a constructor(as it cannot be used to create objects).
